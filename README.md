# Objetos de Aprendizagem — Operações Matemáticas Básicas em Python (Google Colab)

## Descrição do projeto
Nosso projeto consiste em um material didático em formato de notebook Google Colab. O objetivo é ensinar, de forma prática e guiada, a implementação das operações matemáticas básicas em Python: soma, subtração, multiplicação e divisão. O material combina breves explicações teóricas, exemplos executáveis, exercícios e outros materiais para reforço de aprendizado.

## Público-alvo
Este material é recomendado para:
- Alunos do Ensino Fundamental II.
- Estudantes que já tenham contato com operações aritméticas básicas (adição, subtração, multiplicação e divisão).
- Usuários com familiaridade básica em computadores (uso de navegador, executar tarefas simples).

## Requisitos de aprendizagem (objetivos)
Ao final do material, espera-se que o aluno seja capaz de:
1. Entender e saber utilizar as quatro operações básicas implementadas em Python.
2. Manipular corretamente números inteiros em Python.
3. Aplicar operadores aritméticos e reconhecer a precedência entre eles.
